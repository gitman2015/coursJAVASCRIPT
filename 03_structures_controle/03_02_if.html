JavaScript - III/02 - if

If est la plus utilisé des conditions

If

1- if...

La condition if est l'une des plus utilisée et est également la plus simple à appréhender puisqu'elle va nous permettre d'éxecuter un bloc de code si ET seulement si le résultat vaut true.

Attention ! : exceptions

Tout test d'une condition va être évalué dans un contexte booléen; donc à true à l'exception des valeurs suivantes, qui vont être évaluées à false.

  • La valeur 0
  • Le booléen false
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur Nan
